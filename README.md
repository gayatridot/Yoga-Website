# Yoga Website

A frontend-only yoga website built with HTML and CSS, designed to provide information about yoga classes, poses, trainers, and trial courses.

## Project Overview

This is a responsive yoga website that serves as a digital platform for a yoga studio or instructor. The site provides users with information about available classes, yoga poses, instructor profiles, and trial course offerings.

## Technologies Used

- **HTML5** - Semantic markup and page structure
- **CSS3** - Styling and responsive design

## Project Structure

```
Yoga Website/
├── index.html           # Home page - main landing page
├── login.html          # Login page - user authentication interface
├── poses.html          # Yoga poses guide - demonstrations and descriptions
├── trainers.html       # Trainer profiles - instructor information
├── trial-courses.html  # Trial courses - course offerings and registration
├── style.css           # Main stylesheet - all styling and layouts
├── images/             # Image assets directory
└── README.md           # This file
```

## Pages

### 1. **index.html**
The main landing page of the website. Contains the homepage content with navigation, hero section, and key information about the yoga studio.

### 2. **login.html**
User login page. Allows users to authenticate and access their accounts or personalized content.

### 3. **poses.html**
Yoga poses guide page. Displays various yoga poses with descriptions, benefits, and step-by-step instructions.

### 4. **trainers.html**
Trainer profiles page. Showcases information about yoga instructors including their experience, specializations, and certifications.

### 5. **trial-courses.html**
Trial courses page. Features available trial courses and allows users to explore beginner-friendly options.

## Styling

- **style.css** - Main stylesheet containing all CSS rules for layout, colors, typography, and responsiveness across all pages.

## Assets

- **images/** - Directory containing all images and graphics used throughout the website (logos, yoga pose photos etc.)

## How to Use

1. Open `index.html` in your web browser to view the website
2. Navigate between pages using the menu/links on the website
3. All styling is applied through `style.css`
